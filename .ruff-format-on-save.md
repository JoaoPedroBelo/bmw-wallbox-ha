# Auto-format on Save (Recommended)

Ruff will automatically fix blank lines and whitespace when you format.

## Option 1: Format Before Every Commit (Automatic)

```bash
# One-time setup
make pre-commit

# Now every commit will auto-format!
git commit -m "your message"  # Ruff runs automatically
```

## Option 2: Manual Format

```bash
# Format all files
make format

# Or just format specific files
ruff format custom_components/bmw_wallbox/sensor.py
```

## Option 3: Editor Integration

### VS Code
Install the Ruff extension and add to `.vscode/settings.json`:
```json
{
  "[python]": {
    "editor.defaultFormatter": "charliermarsh.ruff",
    "editor.formatOnSave": true
  }
}
```

### PyCharm
1. Install Ruff plugin
2. Settings → Tools → Ruff → Enable "Format on save"

## CI Will Catch Issues

If you forget to format, CI will fail with:
```
❌ Code formatting issues found. Run 'make format' to fix.
```

Then just run `make format` and commit again!
